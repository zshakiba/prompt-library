| Competency      | سؤال                                                                |
| --------------- | ------------------------------------------------------------------- |
| **Delegation**  | آیا نقش AI را مشخص کرده‌ام؟                                         |
| **Description** | آیا خروجی دقیق، مخاطب، زبان و محدودیت‌ها را توضیح داده‌ام؟          |
| **Discernment** | آیا از AI خواسته‌ام پاسخش را نقد، بررسی یا راستی‌آزمایی کند؟        |
| **Diligence**   | آیا از AI خواسته‌ام مسئولانه، شفاف و با رعایت اصول اخلاقی پاسخ دهد؟ |
